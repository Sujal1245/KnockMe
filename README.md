**KnockMe** is a modern Android application that allows users to create and interact with **time-based knock alerts**. Users can create alerts, knock on alerts created by others, and view activity through a clean, reactive interface built using **Jetpack Compose** and **Firebase**.

The project emphasizes **clean architecture**, **unidirectional data flow**, and **modern Android development best practices**, with careful handling of authentication, public user profiles, and efficient Firestore usage.
